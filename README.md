Inventory Management & Stock-Out Analysis

Excel | Pivot Table | Dashboard Project

This project analyzes an inventory dataset using Excel to identify low-stock items, understand stock movement, and support better inventory decisions. The workbook includes a Database sheet, a Pivot Table sheet, and an interactive Dashboard to visualize insights.

✅ Project Overview

The goal of this project is to help businesses avoid stock-outs and maintain healthy inventory levels.
Using Excel functions, pivot tables, and charts, the report highlights critical items that need fast attention.

✅ Dataset Description

The database sheet contains the raw inventory data such as:

Item Name

Category

Current Stock

Minimum Required Stock

Purchase Price

Supplier

Stock In / Stock Out history

This sheet acts as the main source for the pivot table and dashboard.

✅ What I Built
✅ 1. Pivot Table

The pivot table summarizes:

Total stock by item

Fast-moving vs slow-moving products

Items below minimum stock

Category-wise inventory distribution

✅ 2. Dashboard

The Excel dashboard includes:

Bar chart of stock levels

Low-stock alert list

Category-level insights

KPIs (Total Stock, Total Cost, No. of Critical Items)

✅ Key Insights

✔ Identified products that are below minimum stock
✔ Highlighted high-demand and low-demand items
✔ Showed which categories require more purchasing
✔ Created a clean dashboard for quick decision-making
